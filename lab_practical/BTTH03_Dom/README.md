# BTTH03 — HTML, CSS, JavaScript (DOM)

## Cấu trúc

```
btth03_dom/
├── README.md                 ← file này
├── bai1_students/
│   ├── index.html
│   ├── css/style.css
│   └── js/app.js
└── bai2_tasks/
    ├── index.html
    ├── css/style.css
    └── js/app.js
```

## Cách chạy (VS Code)

1. Mở folder `bai1_students` hoặc mở file `bai1_students/index.html`.
2. **Go Live** (hoặc mở trực tiếp file trong trình duyệt).
3. Làm tương tự với `bai2_tasks`.

## Gợi ý học (theo đề thầy)

- Đọc `bai1_students/js/app.js`: các hàm `loadStudents`, `saveStudents`, `renderStudents`, `updateStatistics`, xử lý modal và **event delegation** trên `tbody`.
- Bài về nhà **validation**: bổ sung trong `validateFormBasic()` (Bài 1) hoặc tương tự ở Bài 2 — email, điểm, ngày, độ dài, v.v.

## Lưu ý

- Dữ liệu Bài 1 key: `btth03_students_v1`
- Dữ liệu Bài 2 key: `btth03_tasks_v1`
- Hai bài không dùng chung key localStorage.
